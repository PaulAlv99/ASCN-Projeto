<h2 align="center">
Projeto desenvolvido no âmbito da unidade curricular de Aplicações e Serviços de Computação em Nuvem
do Mestrado em Engenharia Informática na Universidade do Minho
</h2>

<h3 align="center">Sobre o projeto</h3>

<h4 align="center">
Este projeto tem como objetivo criar e automatizar a infraestrutura de uma aplicação utilizando GCP. Foram implementados serviços de monitorização, orquestração
com Kubernetes, automação com Ansible e testes de carga com k6.
</h4>

<h4 align="center">
O projeto inclui configuração de clusters GKE, load balancing, armazenamento de métricas,
dashboards de visualização e testes de performance em diferentes cenários.
</h4>

<h3 align="left">➡️ Funcionalidades implementadas</h3>

<ul>
<li>Deployment automatizado do cluster GKE com Ansible</li>
<li>Orquestração de containers com Kubernetes</li>
<li>Configuração de Load Balancer com a escolha a cair no LB de camada 4 do modelo OSI da Google Cloud</li>
<li>Monitorização com Prometheus (recolha de métricas) e Grafana (visualização)</li>
<li>Configuração de dashboards e datasource Prometheus</li>
<li>Execução de testes de carga e stress tests com k6</li>
<li>Automação de deployment, undeployment e gestão de serviços (AirTrail, monitorização)</li>
<li>Gerenciamento de credenciais e autenticação via GCP CLI (gcloud)</li>
</ul>

<h3 align="left">➡️ Tecnologias utilizadas</h3>

<ul>
<li>Google Cloud Platform (GKE, Load Balancer, VMs, Networking)</li>
<li>Kubernetes</li>
<li>Ansible</li>
<li>Prometheus</li>
<li>Grafana</li>
<li>k6</li>
<li>kubectl</li>
<li>Bash / Shell scripts</li>
</ul>

<h2 align="center">
Project developed for the Cloud Computing Applications and Services course
of the Master's Degree in Computer Science and Engineering at the University of Minho
</h2>

<h3 align="center">About the project</h3>

<h4 align="center">
This project aims to create and automate a cloud-native application infrastructure
using Google Cloud Platform. The system includes monitoring services, container orchestration
with Kubernetes, automation with Ansible, and load/stress testing with k6.
</h4>

<h4 align="center">
The project covers GKE cluster configuration, load balancing, metric storage,
dashboard visualization, and performance testing under different scenarios.
</h4>

<h3 align="left">➡️ Features implemented</h3>

<ul>
<li>Automated deployment of GKE clusters with Ansible</li>
<li>Container orchestration using Kubernetes</li>
<li>Configuration of Google Cloud native Load Balancer L4</li>
<li>Monitoring with Prometheus (metrics collection) and Grafana (visualization)</li>
<li>Dashboards and Prometheus datasource configuration</li>
<li>Execution of load and stress tests with k6</li>
<li>Automation of deployment, undeployment and service management (AirTrail, monitoring)</li>
<li>Credential management and authentication via GCP CLI (gcloud)</li>
</ul>

<h3 align="left">➡️ Technologies used</h3>

<ul>
<li>Google Cloud Platform (GKE, Load Balancer, VMs, Networking)</li>
<li>Kubernetes</li>
<li>Ansible</li>
